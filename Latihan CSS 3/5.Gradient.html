Gradient

1. Linear Gradient

Sintaks sederhana :

background-image : linear-gradient(red, #F0D606,rgba(255,255,0,1), green);


Sintaks Lengkap (sudut) :

background-image: linear-gradient(45deg,red,green);


ARAH :

value :
to bottom : dari atas ke bawah
to top : dari bawah ke atas
to left
to right

kalo diagonal :

to left top : dari kanan bawah ke kiri atas
to left bottom : dari kanan atas ke kiri bawah
to right top
to right bottom

Sintaks Lengkap (arah) :

background-image: linear-gradient(to bottom right,red,green);

Sintaks Color Stop

background-image: linear-gradient(to right, red 63%, green 80%);





2. Radial Gradient

Sintaks singkat :

background-image : radial gradient(red, green,blue);

Sintaks Lengkap :

radial-gradient ([tipe] [jangkauan] at [posisis-y] [posisis-x] [warna 1] [warna 2], ..., ...,[warna n]) ;

Tipe : elipse , circle
Jangkauan : jarak dari pusat gradient ke sisi terluarnya

Cotoh TIPE bulat/circle
