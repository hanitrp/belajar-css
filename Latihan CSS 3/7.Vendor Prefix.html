Vendor Prefix

Pengertian

Vendor : Browser (chrome, firefox, mozila, siri, dll)

Fungsi :

Untuk properti CSS 3 yang tidak support di beberapa browser

Properti

  1. -webkit- (Chrome, Safari, Opera)
  2. -moz- (firefox)
  3. -ms- (Internet explorer)
  4. -o- (opera jadul)
