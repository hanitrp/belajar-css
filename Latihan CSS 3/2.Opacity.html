3.Box-Shadow

1. 4 nilai = box-shadow : x(kanan/kiri) y(atas/bawah) blur color
2. 6 nilai = box-shadow : inset x y blur spreed(size) color
Fungsi Inset : bayangan jadi kedalam

Nilai : 5px 5px 10px red;
box-shadow: 0px 0px 0px 10px pink;
Cara pakai box-shadow biar makin asik :
CARA 1 : Melayang

.enam {
width: 250px;
height: 250px;
background-color: aqua;
margin : 50px auto;
box-shadow:0px 40px 5px -20px rgba(0,0,0,.75);
transition: .3s ;
}

div.enam:hover {
box-shadow: 0px 80px 5px -40px rgba(0,0,0,.4);
margin-top: 40px;
}


CARA 2 : Bayangan dalam kotak

Menggunakan INSET

box-shadow: inset 0 0 10px 10px rgba(0,0,0,.5);

CARA 3 : Multiple Shadow

1. YELLOW
box-shadow: 5px 5px 5px rgba(255, 0, 0,.5), -5px -5px 5px rgba(0,0,255,.5);

2. BLUE : multiple border
box-shadow: 0 0 0 5px red, 0 0 0 10px yellow, 0 0 0 15px magenta;
