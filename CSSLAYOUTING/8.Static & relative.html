Posisition

Static, Relative, Absolute, Fixed




1. STATIC

Jika kita tidak memberi properti posisi pada elemen, maka nilai default dari elemen itu static

2. Relative

Posisinya kaya naik gt kedepan
contoh: si violet



3. Absolute

Ketika kita menggerakan elemen dengam posisi absolute (properti :top, left, bottom, right) , ruang yg ditempati violet dianggap tidak ada dan elemen selanjutnya bakal anggap violet gaada trus diisi

Ketika kita gerakin elemen yg posisi absolute itu, elemen itu akan bergerak relatif thdp posisi elemen parent-nya (elemen body), selama elemen parent juga posisinya non-static

Contoh : si violet (absolute)

container (sibiru) dikasi posisi relative supaya posisinya ikut violet




4.Fixed

Mirip absolute tapii ketika kita mengeerakan elemen dengan posisi fixed, elemen akan bergerak relatif terhadap window(browser)

Dengan kata lain elemen akan terkunci dan tidak bergerak dr posisinya klo halaman discroll
